# @skeleton/types

Shared TypeScript types, interfaces, and enums for the Skeleton monorepo.

Source: [github.com/aryaglekshmi/skeleton-packages](https://github.com/aryaglekshmi/skeleton-packages/tree/main)

---

## Publishing

1. Build the package
   ```bash
   npm run build
   ```

2. Update the version in `package.json`
   ```bash
   npm version patch   # or minor / major
   ```

3. Publish to npm
   ```bash
   npm publish --access public
   ```
